# ORICHALCUM B2B 独立站 — 真实产品数据版 v2.0

## 🏭 项目说明

ORICHALCUM（善同仁科技深圳）B2B独立站，展示4大智能穿戴产品线，支持全球B2B买家在线查阅产品规格、获取报价。

**技术栈**：原生 HTML5 + CSS3 + JS 单页面应用  
**语言支持**：EN / 中文 / ES / AR（四语切换，含阿拉伯语RTL）  
**框架/库**：零依赖

## 🔧 已作的升级

1. **产品数据真实化** — 规格书中的真实芯片型号、传感器、电池容量、尺寸等
2. **外贸专家优化** — MOQ、交期、认证信息在首屏及产品区突出展示
3. **SEO 强化** — 加入了结构化数据（JSON-LD Schema Product/Organization）、hreflang、OG/Twitter Card
4. **表单优化** — 添加公司名必填、产品兴趣选择、Formspree 表单提交
5. **信任信号** — 认证资质、统计数字（5000m²工厂、2M+年产能等）
6. **全产品规格弹窗** — 4种产品 × 4种语言 = 16套规格表
